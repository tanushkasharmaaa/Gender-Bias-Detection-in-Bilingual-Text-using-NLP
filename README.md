Gender Bias Analysis in NLP 🌍🧠
Multilingual Gender Bias Detection using Machine Learning and NLP Techniques

This project explores gender bias in large-scale bilingual text data using Natural Language Processing (NLP), Machine Learning, statistical analysis, and topic modeling techniques.

The system analyzes the IITB English-Hindi Parallel Corpus to detect gender representation patterns, classify gender-oriented text, compute bias scores, and study occupational gender associations across languages.
